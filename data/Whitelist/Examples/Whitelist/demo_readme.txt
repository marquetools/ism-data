The ./Demo-Test-Cases, ./MN-OC-Test-Cases, and ./Other-Test-Cases folders contain
.xml files that will test the whitelist functionality utilizing one of the two
sample configuration files in this folder.  

To test basic whitelist functionality, (test case file names starting with "Whitelist-")
copy the example_whitelist_config.xml to 
../../Schematron/Whitelist/whitelist_config.xml

To test minimum value functionality (test case file names starting with "MinValues-")
copy the example_minvalues_config.xml to 
../../Schematron/Whitelist/whitelist_config.xml

After sample config files have been staged, open the test case files with Oxygen
and validate files.

Description

W3C XML Schema used to facilitate generation of the SchemaGuide for the XML Data Encoding Specification for Multi Audience Collections (MAC.XML).

Introduction

This XML Schema file is only used to produce the schemaGuide for the XML Data Encoding Specification (DES). Please see the document titled XML Data Encoding Specification for Multi Audience Collections for a complete description of the encoding as well as list of all components.

It is envisioned that this schema or its components, as well as other parts of the DES may be overridden for localized implementations. Therefore, permission to use, copy, modify and distribute this XML Schema and the other parts of the DES for any purpose is hereby granted in perpetuity.

Please reference the preceding two paragraphs in all copies or variations. The developers make no representation about the suitability of the schema or DES for any purpose. It is provided "as is" without expressed or implied warranty.

If you modify this XML Schema in any way label your schema as a variant of MAC.XML.

Please direct all questions, bug reports,or suggestions for changes to the points of contact identified in the document referenced above.

Implementation Notes

A MAC.XML is used in conjuction with TDF collections as structured assertions that indicate how objects in a trusted data collection are related. A Trusted Data Collection conforms to the MAC specification when it contains a structured assertion with scope TDC and a multiAudience element. The multiAudienceObject element may also appear to provide additional information about each object in the collection regarding it's relationship to the larger collection. In this way MAC can be used to describe a set of tearlined assets, or other related data sets as needed. Its structure is similar to:

    A tdf:TrustedDataCollection

    • has an assertion mac:MultiAudience
    • optionally has an assertion mac:MultiAudienceObject

&cp_object; v2017-MAY Package Description

&cp_businessObjectClass; (&cp_object;) consists of many files all packaged here for you. This is a simple guide to the layout of this package. Before attempting to follow the links in this guide, please ensure that you have extracted the files from the zip file; otherwise, the links will not work.

As this is the standalone package, none of the dependant specifications will be included and you will have to assemble the full package with an acceptable version of the dependent specifications that you wish to use.

  • Documentation

    This includes the document for this specification and the applicable signature memo associated with it.

    • CSR version 2017-MAY
      • DesCSRXml.pdf
      • SignatureMemo.pdf

  • manifest.md5

    This file contains an MD5 checksum of every file in the package. It can be used to determine if any of the files were corrupted in unzipping or if you have modified a file.

  • CSRpackage.properties

    This file contains information concerning the version, classification, and build time for the package.

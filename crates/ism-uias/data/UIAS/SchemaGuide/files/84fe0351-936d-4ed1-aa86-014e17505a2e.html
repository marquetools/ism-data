Attributes
QName Type Use Annotation
uias:aICP uias:aICPType optional

Reflects whether or not the entity is an AICP
uias:aTOStatus uias:ATOStatusType optional

This attribute indicates the Authority to Operate (ATO) status for the non-person entity
uias:adminOrganization uias:adminOrganizationType optional

Reflects the home organization of the entity.

Contains the token that represents an entity's administrative organization. This is tied to the USAgency Acronym list for US Agencies and follows an open ended regular expression for the other five eyes countries
uias:auditRoutingOrganization uias:auditRoutingOrganizationType optional

Reflects the audit routing organization of the entity.

Contains the token that represents an entity's audit routing organization. This is tied to the CVEnumUIASAuditRoutingOrg CVE which is managed by the IC Audit Subcommittee (ICAS)
uias:authorityCategory uias:authorityCategoryType optional

This attribute specifies the authority(ies) under which the entity is authorized to access and/or discover protected resources
uias:clearance uias:clearanceType optional

Reflects the clearance or classification level of the entity
uias:countryOfAffiliation uias:countryOfAffiliationType optional

Reflects the citizenship(s) or affiliation(s) of the entity
uias:digitalIdentifier uias:digitalIdentifierType optional

Reflects the DN from the entity?s PKI certificate
uias:dutyOrganization uias:dutyOrganizationType optional

Reflects the assigned organization of the entity
uias:dutyOrganizationUnit uias:dutyOrganizationUnitType optional

Reflects the assigned organization unit structure of the entity
uias:entitySecurityMark uias:entitySecurityMarkType optional

Classification and handling of the entity?s digital identifier
uias:entityType uias:entityTypeType optional

Reflects the type of the entity
uias:fineAccessControls uias:fineAccessControlsType optional

Reflects the fine grain access aspects of control systems
uias:group uias:groupType optional

Indicates the group memberships associated with the entity
uias:handlingControls uias:handlingControlsType optional

Indicates the set of handling controls that an NPE is authorized to have
uias:icNetworks uias:icNetworksType optional

List of other IC networks or domains to which an entity?s digital identifier may be transmitted
uias:isICMember uias:isICMemberType optional

Reflects whether or not the entity is a member of the Intelligence Community
uias:lifecycleStatus uias:lifeCycleStatusType optional

Indicates the life cycle phase in which the entity is operating
uias:region uias:regionType optional

Indicates the individual countries or larger sub-regions such as geographical areas of combatant command Areas of Responsibility (AORs), Areas of Interest (AOIs), or State and Non-State Actor(s)
uias:role uias:roleType optional

Indicates the position, job or area of responsibility associated with the entity
uias:topic uias:topicType optional

Indicates the particular intelligence subject area(s) associated with the entity
